## Top 3 pour une location partielle de résidence principale

- Télétransmission EDI-TDFC incluse

- Interface moins moderne

## Points de vigilance

- **Réduction OGA supprimée** (février 2025) : les prix « nets après déduction » affichés par certaines plateformes ne sont plus d'actualité
- **Télétransmission EDI-TDFC** : obligatoire pour la liasse fiscale, vérifier qu'elle est incluse dans le tarif
- **Quote-part** : indispensable si vous louez une partie de votre résidence principale (ex : 2 pièces sur 5)
- **FEC** (Fichier des Écritures Comptables) : obligatoire en cas de contrôle fiscal, vérifier qu'il est généré

## Cas d'usage : location partielle de résidence principale

Si vous louez une partie de votre habitation principale (ex : 2 pièces / 35 m² sur 126 m², soit 27,8 % de quote-part) :

1. **Charges mixtes** : taxe foncière, assurance habitation, intérêts d'emprunt → appliquées au prorata (27,8 %)
2. **Charges dédiées** : travaux dans les pièces louées, mobilier → 100 % déductibles
3. **Amortissements** : immeuble au prorata, mobilier dédié à 100 %
4. **Plafond 120 jours/an** pour les meublés de tourisme en résidence principale

La plateforme choisie doit permettre de définir un **pourcentage d'affectation professionnelle** par charge.
